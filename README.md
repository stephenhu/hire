# hire

Mockups for letshire

## pre-requisites

1.  bootstrap 3.x
1.  jquery 1.8.x
1.  jquery ui themes (specifically the theme named start)
1.  glyphicons
1.  rackup
1.  thin (optional)

## installation

Use a tool like <a href="https://github.io/stephenhu/wrench">wrench</a> to
download all dependencies.

```gem install wrench```
